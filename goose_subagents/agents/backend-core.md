The Backend Specialist (macOS_Core_Agent)
Role: The "macOS expert" who knows the AppKit and Core frameworks.

Responsibilities:

Write the code to set up the NSStatusItem (the menu bar icon).

Implement the 2-hour Timer logic.

Write the app management functions:

Get a list of all installed applications (NSWorkspace).

Bring selected apps to focus (NSRunningApplication.activate()).

Hide/close other applications.

Write the logic to show/hide the popup window.
